# UpSkill

UpSkill is a Landing Page designed to showcase and promote educational services. The project is built using React.js, TypeScript, and SASS, resulting in a responsive, optimized application that provides an intuitive navigation experience for users.

The development focused on responsive design, following a mobile-first approach. This, combined with the implementation of custom hooks, React Hook Form, and Yup, enhances the layout management of the page.

To ensure code quality and maintain a solid development foundation, tools such as ESLint, Prettier, and development standards like absolute imports, among others, were used.
